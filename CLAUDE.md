# llm-wiki

Obsidian-vault, управляемый через Claude Code. Реализует паттерн LLM Wiki: постоянная база знаний, которая обогащается с каждым добавленным источником и каждым заданным вопросом.

## Архитектура

Три уровня:

```
.raw/       # Уровень 1: иммутабельные источники (Claude читает, не изменяет)
wiki/       # Уровень 2: LLM-генерируемая база знаний (Claude владеет)
CLAUDE.md   # Уровень 3: схема и инструкции (этот файл)
```

## Структура wiki/

- `index.md` — мастер-каталог всех страниц, читается первым при запросе
- `log.md` — хронологический журнал операций (append-only, новые записи сверху)
- `cache.md` — кэш недавнего контекста (~500 слов), читается при старте сессии
- `summary.md` — общий обзор базы знаний
- `sources/` — страницы для каждого источника
- `entities/` — люди, организации, продукты, репозитории
- `ideas/` — идеи, концепции, паттерны
- `questions/` — сохранённые ответы на запросы
- `meta/` — дашборды и отчёты

## Конвенции

- Все wiki-страницы содержат YAML frontmatter (плоский, без вложенности)
- Внутренние ссылки через wikilinks: `[[Название]]`
- Имена файлов и папок — на английском, содержимое — на русском
- `.raw/` иммутабельна
- `wiki/log.md` append-only, новые записи сверху
- `wiki/cache.md` перезаписывается полностью каждый раз

## Статус

В разработке. Скиллы добавляются по мере создания.
